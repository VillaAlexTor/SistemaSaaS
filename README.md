# Sistema

Estructura inicial creada:

- `frontend/`
  - `public/index.html`
  - `src/`
    - `components/`
      - `auth/` (Login, Register, Recover)
      - `dashboard/` (Dashboard.jsx)
      - `users/` (UserTable, UserModal)
    - `App.jsx`, `index.js`, `index.css`
  - `package.json`, `tailwind.config.js`
- `backend/` (aquí irá Django más adelante)

Cómo empezar con el frontend (ejemplo con Vite):

1. cd frontend
2. npm install
3. npm run dev

Si quieres, puedo inicializar Vite, configurar ESLint/Prettier, o crear las rutas de React y la integración con Django cuando estés listo.
